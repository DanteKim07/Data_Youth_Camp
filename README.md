# 데이터 소개
> 사용한 데이터들
- 2010-2022년 월별 기온(최저, 최고, 평균)
- 2010-2022 월별 습도(상대, 최저)
- 2010-2022 월별 식중독 환자수
- 2010-2022 월별 약물금액



## 목차
1. 데이터로드 및 전처리(식중독 환자수, 기온, 약물가격 데이터)
2. 간단한 시각화(이상치, 추세그래프)
3. 식중독 환자수 ARIMA 모델 적용, 미래예측
4. 기온 ARIMA모델 적용
5. 약물가격 ARIMA모델 적용, 미래예측
6. 다변량분석(환자수, 기온, 약물가격) VAR모델 적용
